API NeoApp Desarrollo V2

Descripción General

Esta API RESTful permite la gestión de usuarios, autenticación, facturación y notas crédito. Está desarrollada en TypeScript usando Express y MySQL, e integra servicios externos para facturación electrónica.

Estructura del Proyecto

  • controllers/: Lógica de negocio y endpoints.
  • models/: Modelos de datos (People, User).
  • routes/: Definición de rutas Express.
  • utils/: Funciones auxiliares.
  • interfaces/: Interfaces TypeScript.

Guía Rápida de Uso

  1. Instala dependencias: npm install
  2. Configura .env con tus credenciales y datos de conexión.
  3. Ejecuta el servidor: npm run build y npm start
  4. Consulta los endpoints principales:
    • /list-all-user: Listar usuarios
    • /create-user: Crear usuario
    • /sing-in: Inicio de sesión
    • /get-token: Obtener token
    • /create-bill: Crear factura
    • /create-note-credit: Crear nota crédito

Clases Principales

  • People: Modelo base para personas.
  • User: Extiende People, añade atributos y métodos de usuario.
